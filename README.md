This is the Balatro Springfield Mod for Balatro, it is based on the underground indie cartoon "The Simpsons", maybe you've heard of it.
I'm planning on it being a rather large mod when I'm done with it.

Contains 75(kinda) Jokers and 6 enhancements as of release 0.4.5, more are coming and more cards of other aspects also.

Also contains the start of a Texture Pack that loads if you have the Malverk Mod installed, which you can find here: https://github.com/Eremel/Malverk

To install this mod, download the most recent release in the Releases tab, download it and stick it in your mods folder, somewhere in your AppData or something. Check the SMODS guide if you need a hand.
Will eventually release on the Balatro Mod Manager.
